## Tech Stack
Frontend : React, HTML, CSS 
Styles & Design : Tailwind CSS, Lucide Icons
Outils : Git, GitHub
Concepts : Responsive design, animations, composants réutilisables

## Fonctionnalités principales
- Navigation fluide entre les pages
- Animations interactives
- Design responsive pour mobile et desktop
- Composants modulaires et réutilisables

## Installation et lancement local in your project.

1. Cloner le repo : git clone <lien-du-repo>

2. Installer les dépendances : npm install

3. Lancer le projet : npm run dev
